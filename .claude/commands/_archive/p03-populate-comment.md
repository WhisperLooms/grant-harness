# P03: Collaborative Comments

Enable consultant/expert commenting on application (Phase 2).

## Usage
`/p03-populate-comment <company-id> <grant-id> [--reviewer name]`

## Features
- Inline comments on fields
- Suggestion mode (track changes)
- @mentions for specific reviewers
- Comment threads for discussion
- Resolve/approve mechanism

## Reviewer Types
- Company stakeholders
- Grant consultants
- Technical experts
- Financial advisors

## Implementation
NextJS real-time collaboration (Firebase/Supabase)
