# P05: Export Draft Application

Generate draft application document for review (Phase 2).

## Usage
`/p05-populate-export <company-id> <grant-id> [--format pdf|docx]`

## Output
- Professional PDF/Word document
- Matches grant form layout
- Includes all populated fields
- Attachments referenced
- Watermarked "DRAFT"

## Use Cases
- Internal review
- Consultant feedback
- Board approval
- Final proofreading
