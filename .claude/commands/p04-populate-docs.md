# P04: Attach Supporting Documents

Manage document attachments for application (Phase 2).

## Usage
`/p04-populate-docs <company-id> <grant-id> [--upload file1,file2]`

## Document Types
- Financial statements
- Letters of support
- Technical specifications
- Certifications
- Project plans
- Budget spreadsheets

## Tasks
1. Identify required attachments from grant
2. Match available docs from `/c04-company-docs`
3. Flag missing documents
4. Upload and link to application
5. Verify file types, sizes, naming
