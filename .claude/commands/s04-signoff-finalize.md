# S04: Finalize Application

Lock application after all approvals received (Phase 2).

## Usage
`/s04-signoff-finalize <company-id> <grant-id>`

## Tasks
1. Verify all approvals complete
2. Generate final PDF (remove "DRAFT" watermark)
3. Lock application (no more edits)
4. Collect digital signatures
5. Create submission package
6. Archive approved version

## Submission Package
- Final application PDF
- All supporting documents
- Approver signatures
- Submission checklist
- Lodgement instructions

## Output
Ready-to-submit application package
